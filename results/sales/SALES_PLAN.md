# MagicBoxAI Sales Plan

## Positioning
MagicBoxAI is a lightweight AI assistant for task capture, prioritization, and rate-limit safe usage tracking. It targets solo developers, small teams, and operators who need fast input-to-action without heavy setup.

## Target Segments
- Indie developers and founders
- Small product teams (1-10 people)
- Ops and support teams needing quick note-to-task capture

## Core Value Proposition
- Fast: minimal setup, instant API usability
- Safe: built-in rate-limit checks and logging
- Practical: clear endpoints for health, limit check, and save

## Messaging Pillars
1. Ship faster with a tiny AI task capture API
2. Stay within limits and keep audit logs
3. Simple integration, clean docs
