## Execution Flow

### Step 1: Repository Setup

```bash
git clone https://github.com/garyohosu/virtual-company.git
cd virtual-company
```

### Step 2: Configure Git

```bash
git config user.name "Genspark AI Developer"
git config user.email "genspark@virtualcompany.local"
```

### Step 3: Read Task and Previous Results

```bash
cat tasks/CURRENT_TASK.md
cat results/codex/EXECUTION_LOG.md
cat results/gemini/ANALYSIS.md
cat results/claude/INTEGRATION_SUMMARY.md
```

### Step 4: Test Code Execution

```bash
# Example: Test Python code
cd results/codex
python3 [filename].py

# Run any unit tests if provided
python3 -m pytest tests/ -v
```

- Test all components work together
- Verify data flow between modules
- Ensure outputs match expectations
- Validate against success criteria
