#### 1. メインページ：`docs/magicboxai/index.md`

```markdown
# MagicBoxAI - 「動く note」でプログラミングの喜びを

## 何ですか？

MagicBoxAI は、**AI が生成したコードを、そのまま「動かせる」サービス**です。

- HTML コードをコピペ
- ブラウザで即座にプレビュー
- 「保存」ボタンで公開 URL を取得
- 30 日後は自動削除

素人がプログラミングで最初につまずく問題：
- 「index.html.txt になっちゃった」
- 「どうやって動かすの？」
- 「GitHub？メンドクサイ」

**MagicBoxAI なら、その悩みが消えます。**

---

## 使い方（3ステップ）

### Step 1: ChatGPT/Claude に頼む

```
「JavaScript で簡単なゲーム作ってほしい」
「HTML だけで動く、ブロック崩し」
「CSS も使ってね」
```

### Step 2: コードをコピー

ChatGPT がくれた HTML コードをコピー

### Step 3: MagicBoxAI に貼り付け

MagicBoxAI のサイトに貼り付け → **「Save」ボタン**

```
https://magicboxai.example.com/view/abc123def
↑ この URL を友達に送るだけで、誰でも見られる
```

**以上。** プログラミング完成。

---

## 🔐 セキュリティ

- **iframe 隔離**: 悪意あるコードの実行を防止
- **ウイルススキャン**（オプション）
- **30 日自動削除**: 履歴が残らない
- **プロモコード**: 有料版なら保存期間延長

---

## 💰 料金

### 無料版
- 1 日 5 回保存可能
- 30 日で自動削除
- シンプルな機能

### 有料版（プロモコード）
- 無制限保存
- 保存期間無期限
- プレミアム機能
- サポート優先対応

**プロモコードで有料化。** 💳
```
