## 失敗1：Linux と FreeBSD の違いで 2 時間ロス

### 何が起きたか

サーバーを決めずに FastAPI のデプロイ手順を書きました。

```
❌ apt-get（Debian/Ubuntu 向け）
❌ systemd（Linux のみ）
❌ certbot（Linux 向け設定）
```

でも実際は Sakura レンタルサーバー = **FreeBSD**

### 解決策

```
✅ apt-get → pkg
✅ systemd → rc.d
✅ certbot → FreeBSD 版
```

### 学んだこと

**「仮定するな、確認せよ」**

- サーバー環境を最初に確認
- OS を明記する
- 環境特有の手順を別ファイルにする

### 5. CDN の力（1000 字）

```markdown
## CDN の力

MagicBoxAI の UI は豪華です。

```
Tailwind CSS（CDN）→ モダンなデザイン
Alpine.js（CDN）→ インタラクティブ
Font Awesome（CDN）→ 豪華なアイコン
Animate.css（CDN）→ スムーズなアニメーション
```

でも、**バックエンドは PHP + CGI で超軽い**。

### なぜ CDN？

```
本来なら：
HTML/CSS/JS をビルド → ファイル最適化 → デプロイ

CDN なら：
ブラウザが直接読み込む → ビルド不要 → デプロイ簡単
```

### 素人向けの価値

```
MagicBoxAI 自体が「CDN で豪華」だから、
ユーザーは「あ、このくらいなら俺も作れるかも」と思う

心理効果！
```

これがプログラミング教育の最大の工夫です。
```
